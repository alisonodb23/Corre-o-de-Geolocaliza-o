Geolocalização de IP Incorreta

Guia para orientar clientes cujo conteúdo de streaming ou canais aparece como se estivessem em outro estado.

Ver Roteiro de Diálogo Ver Passo a Passo

Diálogo com o Cliente

Siga estes passos para um atendimento claro e eficiente.

Passo 1: Identificar e Explicar a Causa

Valide o problema do cliente e explique a causa de forma simples.

"Olá, Sr(a). [Nome do Cliente]. Entendo que os canais do(a) senhor(a) ou o serviço de streaming estão exibindo a programação de outro estado. Isso é algo que podemos resolver."

"Isso geralmente acontece por um motivo simples: o 'mapa' que os serviços de streaming usam para localizar sua conexão na internet às vezes tem uma informação desatualizada para o seu endereço de IP. Nosso trabalho será pedir a eles que corrijam essa informação no mapa."

Passo 2: Iniciar o Processo de Correção

Peça a colaboração do cliente para obter a informação necessária: o endereço IP público dele.

"Para podermos solicitar a correção, primeiro precisamos saber qual é o seu endereço de IP atual. O(A) senhor(a) poderia, por favor, acessar um site rápido para mim em seu computador ou celular?"

Passo a Passo Detalhado da Correção

Guie o cliente (ou execute você mesmo) através deste processo.

1. Obter o Endereço IP do Cliente

Peça ao cliente para acessar o site www.meuip.com.br e informar o número que aparece em destaque.

Anote o endereço IP informado pelo cliente.

2. Acessar a Ferramenta de Correção MaxMind

Com o IP em mãos, acesse (ou peça para o cliente acessar) o formulário de correção no site da MaxMind.

Abrir Ferramenta de Correção MaxMind

3. Preencher e Enviar o Formulário

Preencha os campos do formulário com as informações corretas do cliente. Os campos obrigatórios geralmente são:

  • IP Address: Cole o IP que o cliente informou.
  • Country: Brasil
  • State/Province: O estado correto do cliente.
  • City: A cidade correta do cliente.
  • Postal Code: O CEP correto do cliente.

Após preencher, envie o formulário.

4. Alinhar Expectativas

É crucial informar ao cliente que a mudança não é imediata.

"Pronto, Sr(a). [Nome do Cliente], a solicitação para corrigir a localização do seu IP foi enviada com sucesso. É muito importante que o(a) senhor(a) saiba que essa atualização pode levar alguns dias, ou em alguns casos, até uma ou duas semanas para ser processada por todos os serviços de streaming. Por favor, acompanhe nos próximos dias para ver se a programação já foi corrigida."

Perguntas Frequentes (Para o Atendente)

Se houver resistência, você pode obter o IP do cliente através de um acesso remoto via anydesk (se disponível) e preencher o formulário da MaxMind em nome dele. Apenas confirme os dados de cidade e estado para garantir a precisão.

Pode resolver temporariamente. Como os IPs são dinâmicos, ao reiniciar o modem o cliente pode receber um novo IP que tenha a geolocalização correta. Você pode sugerir isso como uma tentativa rápida, mas informe que a correção definitiva para o IP problemático (que pode voltar para ele no futuro) é feita pelo site da MaxMind.
